## 摘要
多模态大语言模型必须适应不断变化的任务和领域，但在受限部署足迹下持续改进仍然困难，因为重复的参数更新或不断增长的记忆回放存储会随时间累积适应状态。我们研究固定足迹的持续适应：部署的适应状态保持在固定记忆预算下，同时骨干模型保持不变，任务特定更新被外部化。我们提出InduceKV，一种基于检索的方法，将每个选定的训练前缀存储为注意力就绪的记忆条目，包括冻结的检索键和紧凑的逐层键值（KV）载荷，这些载荷可以附加到模型的自注意力缓存中。在严格的记忆预算下，InduceKV通过双层选择构建紧凑的诱导集：轻量级校准用于检索，而选中的记忆平衡当前任务似然、基于锚点的保留以及冻结检索空间中的覆盖。在任务增量指令微调、持续VQA、领域增量适应和终身多模态指令微调中，InduceKV在匹配的记忆预算下持续优于PEFT、MoE、回放和提示检索基线。我们进一步报告了骨干匹配、阶段1 CoIN、计算匹配和可扩展性诊断，表明收益并非来自更强的骨干、仅回放或无限候选池。

## Abstract
Multimodal large language models must adapt to evolving tasks and domains, yet continual improvement under bounded deployment footprint remains difficult because repeated parameter updates or growing replay stores can accumulate adaptation state over time. We study fixed-footprint continual adaptation: the deployed adaptation state is kept under a fixed memory budget, while the backbone model is left unchanged and task-specific updates are externalized. We propose InduceKV, a retrieval-based method that stores each selected training prefix as an attention-ready memory entry, consisting of a frozen retrieval key and compact layerwise key--value (KV) payloads that can be appended to the model's self-attention cache. Under a strict memory budget, InduceKV constructs a compact inducing set through bilevel selection: a lightweight calibration is fit for retrieval, while the selected memory balances current-task likelihood, anchor-based retention, and coverage in the frozen retrieval space. Across task-incremental instruction tuning, continual VQA, domain-incremental adaptation, and lifelong multimodal instruction tuning, InduceKV consistently improves over PEFT, MoE, replay, and prompt-retrieval baselines under matched memory budgets. We further report backbone-matched, stage-1 CoIN, compute-matched, and scalability diagnostics, showing that the gains are not due to a stronger backbone, replay alone, or an unbounded candidate pool.

### 1. 核心问题与整体含义（研究动机与背景）

-   **核心问题**：多模态大语言模型（MLLMs）在部署后需要不断适应新任务和新领域（持续适应）。然而，现有方法存在一个关键矛盾：传统的参数更新方法（如PEFT、MoE）会导致适应状态不断累积，难以满足固定的、有限的部署内存预算；而基于回放的方法则面临存储历史数据与维持固定内存预算之间的冲突。论文旨在解决**如何在完全冻结骨干模型、且部署的外部内存预算严格固定的前提下，实现有效的持续适应**。
-   **研究背景**：当前主流持续适应方法（PEFT、MoE、重放）要么重写模型参数，要么依赖训练时数据管理，其稳定-可塑性权衡受限于参数空间或数据存储，难以在固定、小内存空间下高效运作。论文认为，理想的方式是将新信息注入推理路径，但不改变模型参数。

### 2. 方法论

-   **核心思想**：将持续适应问题重新定义为预算约束下的在线诱导集（Inducing Set）选择问题。通过冻结MLLM的骨干网络，将任务增量外部化为紧凑的、可直接被注意力机制调用的键值（KV）记忆。模型在推理时，通过检索这些外部KV记忆并将其注入自注意力缓存来影响生成，无需任何梯度更新。
-   **关键技术细节**：
    1.  **记忆条目提取**：对于每个训练前缀，从冻结的MLLM中提取：
        -   一个**检索键**：对最终层隐藏状态进行均值池化和L2归一化，用于余弦相似度检索。
        -   一组**紧凑的逐层KV负载**：对每层的原始键和值进行分段均值池化，得到固定长度（m）的压缩表示。
    2.  **检索与注入**：
        -   **检索**：对于测试前缀，计算其检索键，并与内存中所有记忆条目的检索键进行余弦相似度匹配。
        -   **注入**：检索到的记忆条目的KV负载，经过**检索权重**（基于相似度，受温度参数τ控制）和**逐层的值门控**（λℓ，控制每层内存影响强度）加权后，拼接到标准自注意力缓存的KV序列中，作为额外的缓存token参与后续的解码过程。校准参数ϕ = {τ, λ1,..., λL}非常轻量。
    3.  **双层诱导集选择**：在固定预算B下，通过一个双层优化框架来选择最有价值的记忆条目（诱导集）：
        -   **内层问题**：在给定候选池和选择权重w后，通过最小化当前任务NLL来拟合（更新）轻量级校准参数ϕ*(w)，确保检索和注入功能在当前任务上有效。
        -   **外层问题**：优化选择权重w，使其在预算内。目标函数包含三项：
            -   **当前任务拟合**：最小化当前任务的NLL。
            -   **历史保留（锚点）**：最小化在一个小规模历史锚点集（A<t）上的NLL，以避免遗忘。
            -   **覆盖率**：通过一个**频谱覆盖正则化器**（对加权协方差矩阵的行列式取对数）来惩罚冗余的、高度相似的检索键，鼓励记忆的多样性。
        -   优化完成后，根据权重w选择Top-B个条目作为最终的部署内存Mt。

### 6. 主要结论与发现

-   **InduceKV方法有效**：在固定内存预算下，在所有测试的持续适应场景中，InduceKV一致且显著地优于PEFT、MoE、重放和提示检索基线。
-   **外部化任务更新可行**：通过冻结骨干并构建紧凑的、可检索的KV外部记忆，可以实现有效的持续适应，性能和遗忘控制均优于参数更新方法。
-   **双层选择是关键**：消融实验表明，内层校准、锚点保留、频谱覆盖和检索加权注入等组件都贡献了性能提升，而双层耦合优化是必要的。
-   **增益来源**：性能提升主要源于**经过精心选择的、非冗余的、与注意力兼容的外部记忆**，而非更强的骨干、简单的重放或无限的候选池。模型在需要时会主动“路由”到检索到的KV负载上。

### 7. 优点

-   **问题重构巧妙**：将持续适应问题转化为一个具有理论支持的、清晰的预算在线子集选择问题，为稳定性-可塑性张力提供了一个新的视角。
-   **方法新颖高效**：提出了一种全新的“外部化+检索+注入”的适应范式，通过紧凑的KV负载和轻量校准，避免了冗余的模型参数更新。
-   **理论分析扎实**：提供了在线凸优化框架下的遗憾界分析，证明了方法在理论上具有次线性遗憾和动态适应性。
-   **实验极其充分和公平**：实验覆盖了多种主流持续学习设定，对比基准全面且公平匹配了预算和计算量，消融实验深入，机制分析透彻，充分验证了方法的有效性和设计合理性。

### 8. 不足与局限

-   **推理开销权衡**：并非零开销方法。与无外部记忆的PEFT相比，其预填充计算成本更高，因为它需要额外的前向传递来生成检索键。论文明确指出这是**计算感知（compute-aware）**的方法，而不是零开销替代品。
-   **依赖模型架构**：当前实现需要访问冻结MLLM的隐藏状态和KV缓存接口。对于使用自定义注意力实现或受限推理API的模型族，可能需要额外的工程适配。
-   **锚点近似风险**：理论分析依赖锚点损失能精确近似历史风险的假设。虽然实验验证了这一假设在测试场景中成立，但其在更复杂或者分布偏移剧烈的场景下的鲁棒性有待进一步验证。
-   **知识路由的简单性**：检索基于简单的余弦相似度，注入通过网络顶部的加权平均实现。这种“软路由”可能不如MoE的“硬路由”或专家选择那样具有细粒度控制能力。处理需要精确组合多个来源知识的任务时，其表现可能受限于此。
-   **可解释性**：虽然论文测量了注意力质量，但模型的决策过程在很大程度上仍然是一个黑盒。注入的KV记忆是如何被组合并用于生成特定 token 的，其内部机制未被深入解释。
